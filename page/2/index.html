简述渲染管线
发表于2021-08-01|Render
  渲染管线主要分为三个阶段：应用程序阶段、几何阶段、光栅化阶段。 应用程序阶段（CPU负责）：主要和CPU、内存打交道，诸如碰撞检测、场景图建立、视锥裁剪等算法在此阶段进行。在该阶段的最后，几何体数据（顶点坐标、法向量、纹理坐标等）通过数据总线传送到图形硬件。 几何阶段（GPU负责）：主要负责顶点坐标变换、光照、裁剪、投影以及屏幕映射。在该阶段的最后，得到经过变换和投影之后的顶点坐标、颜色、纹理坐标。 光栅化阶段（GPU负责）：基于几何阶段输出的数据，为像素配色，最终得到最终渲染的图像。 几何阶段 从 object space 到 world space  object space coordinate 与其他物体没有任何参照关系，注意，这个概念非常重要，它是将 object space coordinate 和 worldspace coordinate 区分开来的关键。无论在现实世界，还是在计算机的虚拟空间中，物体都必须和一个固定的坐标原点进行参照才能确定自己所在的位置，这是world space coordinate的实际意义所在。 ...
